# **App Name**: PAUL

## Core Features:

- Product Catalog: Browse and filter bags, umbrellas, apparels and accessories by type, material, and price.
- Product Details: View product descriptions, high-resolution images, sizes, and care instructions.
- Shopping Cart: Add, remove, and modify items in a shopping cart.
- Checkout Process: Securely process payments and manage shipping addresses.
- Order History: View past orders and track shipment statuses.
- AI Style Tool: An AI-powered "style tool" suggests apparel and accessories based on the selected product and user preferences using a trained model that recommends the best combinations, enhancing the shopping experience.
- User Accounts: Secure user authentication and profile management.

## Style Guidelines:

- Primary color: Charcoal gray (#333333) for a premium and modern feel.
- Background color: Off-white (#F5F5F5) to provide a subtle contrast against content, establishing a clean backdrop that enhances visual elements without overpowering them.
- Accent color: Cool gray (#888888) for subtle UI elements and accents.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral and objective feel.
- Use minimalist, custom-designed icons.
- Minimalist layout with generous spacing and clean lines to provide a luxury experience.
- Subtle transitions and animations for a refined user experience.
